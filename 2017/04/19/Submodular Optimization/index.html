Submodular Optimization：概念、算法、加速

发表于 2017-04-19

概念

1) 什么是Set Function？
维基百科的定义如下：

In mathematics, a set function is a function whose input is a set. The output is usually a number. Often the input is a set of real numbers, a set of points in Euclidean space, or a set of points in some measure space.

简言之，集合函数将一个集合映射为一个实数值。

2) 什么是Submodular Set Function？

Let $N$ be a finite set and $z$ be a real-valued function defined on the set of subsets of $N$ that satisfies $z(S)+z(T)\geq z(S\cup T)+z(S\cap T)$ for all $S$, $T$ in $N$. Such a function is called submodular.

重述一下：$N$是一个有限集，$z$是一个定义在$N$的子集上的实值函数，如果对于$N$的所有子集$S$和$T$, $z$都满足$z(S)+z(T)\geq z(S\cup T)+z(S\cap T)$，那么就说该函数是次模（Submodular）的。

对于$z$需要满足的条件有另外一种等价的表述：
$$
z(S\cup {k})-z(S)\leq z(R\cup {k})-z(R), ~~R\subset S\subset N, ~~k\in N-S \tag{1}
$$
     $(1)$称为diminishing returns property。为什么叫“回报减小”呢？举例。

进一步说明，
$$
z(S\cup {k})-z(S)\geq 0, ~~S\subset N, ~~k\in N-S \tag{2}
$$
定义在$N$的子集上的实值函数$z(S)$满足$(1)$叫做submodular set function；同时满足$(1)$和$(2)$则叫做submodular nondecreasing set function。

另外，如果$z(\emptyset)=0$, $z$是normalized的。

3) 什么是Submodular Optimization？
$$
\max_{S\subseteq N}{z(S):\left|S\right|\leq K, z(S) ~\rm{submodular} } \tag{3}
$$
找到$N$的一个子集$S$, $S$的基数（ Cardinality，即$S$中元素个数）不大于$K$，并使得$z(S)$取得最大值。

变种
$$
\max_{S\subseteq N}{z(S):c(S)\leq K, z(S) ~\rm{submodular} }
$$

$$
c(S)=\sum_{s\in S}c(s)
$$

算法

A natural way to find solutions to problem $(3)$ quickly is to start from the null set and add elements one at a time, taking at each step that element which increases $z$ the most.

marginal gain:
$$
\rho_{j}(S)=z(S\cup{j})-z(S)
$$
1) 针对submodular nondecreasing set function的情况：
Let $S^0=\emptyset$, and iteratively, in step $t$, adds $i$ which maximizes the marginal gain $i(t)=\underset{i\in N\setminus S^{t-1}}{\arg\max}\rho_{i}(S)$, set $S^{t-1}=S^{t-1}\cup{i(t)}$.
The algorithm stops, once it has selected $K$ elements.

2) 针对submodular set function的情况：
Initialization. Let $S^0=\emptyset$, $N^0=N$ and set $t=1$.
Iteration t. Select $i(t)\in N^{t-1}$ for which $\rho_{i(t)}(S^{t-1})=\max_{i\in N^{t-1}}\rho_{i}(S^{t-1})$ with ties settled arbitrarily. Set $\rho_{t-1}=\rho_{i(t)}(S^{t-1})$.
​ Step 1. If $\rho_{t-1}\leq0$, stop with the set $S^{K^*}$, $K^*=t-1<K$. If $\rho_{t-1}\gt0$, set $S^{t-1}=S^{t-1}\cup{i(t)}$ and $N^t=N^{t-1}-{i(t)}$. Continue.
​ Step 2. If $t=K$, stop with the set $S^{K^*}$, $K^*=K$. Otherwise set $t\rightarrow t+1$.

Submodular Optimization是NP-Hard的，Greedy Algorithm是一个近似解法。

For example, when $z(S)$ is nondecreasing and $z(\emptyset)=0$, we show that a “greedy” heuristic always produces a solution whose value is at least $1-{[(K-1)/K]}^K$ times the optimal value. This bound can be achieved for each $K$ and has a limiting value of $(e-1)/e$, where $e$ is the base of the natural logarithm.
